The Logic: ROI over Vanity

pytest-coverage-impact

is a plugin that brings ROI (Return on Investment) logic to your test suite. It parses your code's Abstract Syntax Tree (AST), builds a call graph, and uses Machine Learning to score every function in your codebase based on two factors:

Impact (The Blast Radius):

How many other functions depend on this? If

process_payment()

breaks, the app dies. If

get_pretty_name()

breaks, a log looks ugly.

Complexity (The Friction):

How hard will it be to write a test for this?

Priority Score

using this formula:

This score gives me (and my AI assistant) a prioritized hit-list. Instead of asking Cursor to вЂњincrease coverage,вЂќ I can paste the top 5 functions from the Impact Report and say,

вЂњWrite tests for these specific functions.вЂќ

Under the Hood: Machine Learning for Code

Most tools measure вЂњcomplexityвЂќ by just counting nested if-statements (Cyclomatic Complexity). I wanted something smarter.

I trained a Random Forest model to predict

вЂњTest Pain.вЂќ

The model analyzes static code features to guess how annoying a function will be to test:

Side Effects:

Does it touch the filesystem, network, or external APIs (like Snowflake)?

Test Anatomy:

Does testing this usually require mocks, fixtures, or integration markers?

The tool predicts вЂ” on a scale of 0.0 to 1.0 вЂ” how complex the testing effort will be.

High Priority:

High Impact + Low Complexity (The вЂњMust HavesвЂќ)

Low Priority:

Low Impact + High Complexity (The вЂњTime SinksвЂќ)

It even calculates the

variance

between the trees in the Random Forest. If the model isnвЂ™t confident in its prediction (wide confidence interval), it automatically deprioritizes that function. We only want to chase sure things.

The Killer Feature: Train It on YOUR Code

Every codebase is different. What counts as вЂњcomplexвЂќ in a monolithic web app is different from a Data Engineering repo.

I included a training pipeline directly into the CLI so the tool can learn from

Bash

This command maps your existing functions to their tests, extracts the features, and trains a bespoke Random Forest model (
